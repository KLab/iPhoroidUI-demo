■ KLab iPhoroid UI 概要

 KLab iPhoroid UI は Android 2.2 以上の Android 環境で動作する UI ライブラリのセットと
 それを用いたデモプログラムです。次のような機能があります。 

 1. リストビュー
    - twitter ライクなページング機能
    - 無限サイズのリストを扱える画像メモリ管理
 2. グリッドビュー
    - リストビューと同等に扱えるアダプター
    - 無限サイズのリストを扱える画像メモリ管理
 3. フロービュー
    - リストビューと同等に扱えるアダプター
    - 無限サイズのリストを扱える画像メモリ管理
 4. カバーフロー
    - リストビューと同等に扱えるアダプター
    - 無限サイズのリストを扱える画像メモリ管理

 KLab iPhoroid UI の動作は以下のテスト環境においてのみ確認を行っています。

  Android 2.2 が開発できる環境

  確認済みの開発環境

    Eclipse 3.7                http://www.eclipse.org/downloads/
    ADT 16.0.1                 http://developer.android.com/sdk/index.html
  	
■ デモアプリインストールとアンインストールの方法 

  デモアプリ UI_DEMO のインストール
  
	Android 端末の SD カードの /mnt/sdcard/iphoroid/data1 フォルダに
	ui_demo.zip 中の tmp/iphoroid/data1/*.jpg を実行前にコピーしておいてください。
	
	USB 端子を PC に接続して
	
		$ your_android_adk_dir/platfrom_tools/adb install UI_DEMO.apk

  アンインストール
  
  	Android 端末のアプリケーション管理から uninstall してください。
